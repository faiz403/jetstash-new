# JetStash brand photography

Drop correctly named files here and deploy — nothing else is required.

```
heroes/<key>.jpg          page hero backdrops
destinations/<slug>.jpg   one per destination
airports/<slug>.jpg       one per UK airport
guides/<slug>.jpg         one per guide (optional tier)
```

Accepted: `.avif` `.webp` `.jpg` `.jpeg` `.png` (that priority order).

The complete shot list — subject, landmark, aircraft position, lighting,
composition, AI prompt and alt text for every single image — lives in
**`docs/visual-identity.md`**. Follow it exactly; that document is what keeps
fifty images produced months apart looking like one brand.

Until a file exists, the site renders its generated on-brand panel — nothing
ever looks broken. Delete a file to revert.
